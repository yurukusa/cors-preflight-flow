CORS Decision Rules

ConditionResult
Same-origin requestNo CORS needed
Cross-origin GET/HEAD/POST with only safe headersSimple Request
PUT / DELETE / PATCHPreflight Required
Custom headers (e.g. Authorization, X-Custom)Preflight Required
Content-Type other than text/plain, multipart/form-data, application/x-www-form-urlencodedPreflight Required
Server doesn't return Access-Control-Allow-OriginBlocked
credentials: 'include' + Access-Control-Allow-Origin: *Blocked (wildcard not allowed with credentials)
